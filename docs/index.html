Getting Started

Installation

Installation with RCloud

Run the following using the rcloud shell: 

rcloud app install librstd_dev

Compilation

Compiling Using RCloud Pkg-Config Tool


g++ `rcloud pkg-config include librstd_dev` -o myapp main.cc `rcloud pkg-config libs librstd_dev`

Compiling Using Muon


muon c++ -s *.cc -l librstd_dev -o myapp

Using RStd


#include <RStd/RStd.h>
using namespace RStd;
